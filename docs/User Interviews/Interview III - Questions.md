# Interview III - Questions

**Important:** Don't introduce what the app does before the person gives comments first.

1. What would you do first here? *(as soon as person opens the app)*
2. What feels confusing or unnecessary?
3. In one sentence, what do you think this app does?
4. What is the ONE feature you would definitely use?
5. What feels like too much effort?
6. If you had to delete one feature, what would it be?
7. If you had to add one feature, what would it be?
8. Do you have any comments on visual design?
