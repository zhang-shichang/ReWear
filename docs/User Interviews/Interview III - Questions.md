# Interview III - Questions

**Important:** Don't introduce what the app does before the person gives comments first.

1. What would you do first here? *(as soon as person opens the app)*
2. What do you think the flow of the application is, and what should be a reasonable flow?
3. What feels confusing or unnecessary?
4. In one sentence, what do you think this app does?
5. What is the ONE feature you would definitely use?
6. What feels like too much effort?
7. How often would you use it?
8. If you had to delete/modify one feature, what would it be?
9. If you had to add one feature, what would it be?
10. Do you have any comments on visual design?
